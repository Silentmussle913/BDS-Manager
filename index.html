BDS-Manager with Playit Protocol

The Ultimate Bedrock Dedicated Server Manager

About the Project

The BDS-Manager is a lightweight and efficient tool for managing Bedrock Dedicated Servers with the Playit protocol built-in for seamless tunneling. It simplifies server hosting for Minecraft Bedrock Edition, allowing you to focus on creating a great gaming experience.

Features

  • Easy server setup and management
  • Integrated Playit protocol for automatic tunneling
  • Customizable configurations
  • Efficient resource management

How to Use

To get started with the BDS-Manager:

  1. Clone the repository from the link below.
  2. Place your server files into the server folder.
  3. Run start.bat to launch the manager.
